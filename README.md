
<h1>Super Trunfo 🏆</h1>
<p>Status: Finalizado</p>
  
<h2>Descrição</h2>
<p>O Super Trunfo é um jogo desenvolvido em C, seguindo a dinâmica clássica onde os jogadores competem utilizando cartas com atributos variados. </p

<h2>Como jogar 🎮</h2>
<ol>
  <li>
    Cadastre duas cartas diferentes.
  </li>
  
  <li>
    Escolha dois atributos como desempate.
  </li>
  
  <li>
    Vence primeiro a cidade com menos densidade populacional, caso haja empate, o jogo utiliza os dois criterios de desempate escolhidos pelos jogadores para definir um vencedor.  
  </li>
</ol>

<h2>Regras</h2>
<ul>
  <li>
      Os jogadores cadastram duas cartas
  </li>
  
  <li>
      Na seleção do atributo de desempate, não é permitido escolher duas vezes o mesmo atributo
  </li>
</ul>
